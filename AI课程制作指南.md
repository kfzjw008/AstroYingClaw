## 🚀 快速开始

### 第一步：准备好基础信息
在开始对话前，准备好以下信息：
- 课程主题（如：春季星空、行星观测）
- 目标时长（10-60分钟）
- 目标观众（小学生/中学生/成人）
- 重点天体或星座

### 第二步：使用标准化提问模板
直接复制下面的模板，修改括号中的内容即可。

## 💬 提题模板

### 模板1：创建新课程（最简单）

```
请帮我创建一个天文教学课程，主题是【课程主题】，

要求：
1. 包含3-5个主要天体
2. 每个天体讲解后播放语音
```

## 🔄 对话流程

### 标准对话流程

#### 第1轮对话：创建课程大纲
**你：**
```
请帮我创建一个关于夏季星空的入门课程，
20分钟长，适合初中生。
```

**AI会返回：**
- 课程大纲
- 建议包含的天体
- 时间分配建议

#### 第2轮对话：生成流水线配置
**你：**
```
很好！现在请帮我生成对应的流水线配置文件，
使用YAML格式，包含以下步骤：
1. 课程介绍语音
2. 移动到第一个天体
3. 开启激光指示
4. 播放讲解语音
5. 重复2-4步骤（对每个天体）
6. 课程总结
7. 停放设备
```

#### 第3轮对话：验证和完善
**你：**
```
请检查生成的配置文件：
1. 时间计算是否正确？
2. 串口指令格式是否正确？
3. 是否遗漏了任何重要步骤？
4. 激光安全延迟是否已设置？
```

#### 第4轮对话：生成配套文件
**你：**
```
现在请帮我生成：
1. 课程的讲解语音文本（用于TTS）
2. 课程的描述文档
3. 测试命令
```

## 📝 示例对话

### 完整示例：创建"秋季星空"课程

#### 第1轮：确定课程内容
**你：**
```
请帮我创建一个秋季星空观测课程，25分钟，适合中学生。
```

**AI：**
```
好的！我建议创建一个25分钟的秋季星空课程，包含：

1. 课程导入（2分钟）
   - 秋季星空特点介绍
   - 观测注意事项

2. 主要观测对象（20分钟）
   - 仙后座（W形星座）
   - 飞马座和仙女座
   - 英仙座和英仙座流星雨
   - 秋季四边形

3. 课程总结（3分钟）
   - 知识回顾
   - 鼓励继续探索

是否要我开始生成详细的流水线配置？
```

#### 第2轮：生成配置文件
**你：**
```
是的，请生成完整的YAML流水线配置文件。
要求：
- 每个星座讲解前先移动指星仪
- 开启激光指示10秒
- 播放讲解语音
- 串口使用COM3，波特率9600
```

**AI：**
```
# 生成的流水线配置（实际会返回完整YAML）
name: "秋季星空观测"
description: "认识秋季夜空的主要星座"
duration: 25
...
```

#### 第3轮：生成音频文本
**你：**
```
很好！现在请为这个课程生成讲解语音的文本，
每段10-30秒，适合TTS转换为语音文件。
```

**AI：**
```
# 讲解语音文本

## intro.txt
欢迎来到秋季星空观测课程。在接下来的25分钟里，
我们将一起探索秋季夜空中最壮观的星座...

## cassiopeia.txt
现在让我们看向北方，寻找一个W形的星座...
```

### 技巧2：指定约束条件
```
请创建一个课程，要求：
- 课程时长在15-30分钟之间
- 最多包含5个观测对象
- 每个对象讲解时间不超过3分钟
- 必须包含激光安全措施
- 必须有课程开始和结束语音
```

### 技巧3：参考现有课程
```
请参考lesson-01-summer-stars.yaml的格式，
帮我创建一个类似结构的春季星空课程，
重点观测狮子座、室女座和牧夫座。
```

### 技巧4：多轮迭代优化
```
# 第1轮
创建一个课程...

# 第2轮（反馈后）
请把课程的第3步和第4步合并，
并增加5秒的等待时间。

# 第3轮（再优化）
请检查时间计算是否正确，
并在每个讲解前添加"请看"的提示语。
```

## 🔧 实用操作步骤

### 步骤1：准备阶段
```bash
# 1. 创建课程目录
mkdir -p courses/autumn-stars

# 2. 复制模板
cp pipelines/template.yaml courses/autumn-stars/
```

### 步骤2：对话生成
1. 打开AI对话界面（OpenClaw聊天、ChatGPT等）
2. 使用上面的提问模板
3. 保存AI生成的配置文件

### 步骤3：验证配置
```bash
# 验证生成的配置
openclaw astro validate courses/autumn-stars/course.yaml

# 测试运行（模拟模式）
openclaw astro start courses/autumn-stars/course.yaml --dry-run
```

### 步骤4：生成音频
```
# 在AI对话中继续提问：
"请为上面的课程生成TTS语音文本，每段10-20秒，
使用温馨亲切的语气，适合中学生。"

# 使用TTS工具生成音频文件
# Windows: 使用PowerShell
# macOS: 使用say命令
# Linux: 使用espeak
```

### 步骤5：测试运行
```bash
# 正式运行
openclaw astro start courses/autumn-stars/course.yaml

# 监控状态
openclaw astro status
```

## ⚠️ 常见问题

### Q1：AI生成的配置有错误怎么办？
**A：** 提示AI检查和修复：
```
请检查以下配置文件是否有错误：
【粘贴配置文件】

特别注意：
1. 时间是否超出总时长
2. 串口指令格式是否正确
3. 文件路径是否正确
4. YAML缩进是否正确
```

### Q2：如何让生成的课程更生动？
**A：** 在提问中添加要求：
```
请创建一个生动有趣的课程，要求：
- 使用比喻和类比
- 加入互动问题
- 添加趣味小知识
- 语言风格适合【目标年龄】
```

### Q3：如何生成配套资料？
**A：** 多步骤提问：
```
# 第1步：生成课程脚本
请为这个课程生成详细的讲解脚本...

# 第2步：生成问答
请为这个课程设计5个测试题...

# 第3步：生成扩展阅读
请推荐3个相关的学习资源...
```

### Q4：如何调整已有课程？
**A：** 参考式提问：
```
请基于lesson-01-summer-stars.yaml，
帮我修改：
1. 把"夏季大三角"换成"冬季大三角"
2. 增加5分钟的天狼星讲解

```
