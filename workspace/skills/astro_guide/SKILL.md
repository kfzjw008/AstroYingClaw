# 🔭 AstroGuide - 智慧天文教学自动化系统

## 技能概述

AstroGuide 是一个面向天文教育的全流程自动化系统。当用户询问星空、星座、天体等相关问题时，本技能提供：

### 三大核心能力

1. **自动化流水线执行** - 按预设时间表自动控制硬件+播放语音，实现无人值守教学
2. **课件自动生成** - 自动生成 Markdown 讲稿（2000字）+ PowerPoint 课件（4-5页）
3. **实时硬件控制** - 通过串口控制激光指星仪指向目标天体，同步播放语音解说

### 核心组件

#### 1. 自动化流水线系统

流水线是一个 YAML/JSON 文件，描述完整的教学流程：

```yaml
name: "双子座深度观测课程"
duration: "15min"
mode: "auto"  # auto=自动执行, manual=手动确认每步
steps:
  - time: "00:00"
    action: "point"
    target: "双子座_北河三"
    text: "让我们从双子座最亮的恒星北河三开始..."

  - time: "00:45"
    action: "speak"
    text: "北河三距离地球约34光年，是一颗橙巨星..."

  - time: "02:30"
    action: "point"
    target: "双子座_M35星团"
    text: "接下来让我们观测M35疏散星团..."
```

#### 2. 天体数据库

内置中文天体名称映射表，支持：
- 88个星座（完整中文+拉丁名称+边界坐标）
- 亮度前500的恒星（含中文名、拜耳编号、坐标）
- 梅西耶天体（110个深空天体）
- 常见太阳系天体

#### 3. 流水线生成工具

从自然语言描述自动生成流水线文件：
```
生成一个关于猎户座的教学流水线，包含参宿四、参宿七、猎户座大星云
```

## 工具使用指南

### 工具 1: run_pipeline - 执行自动化流水线

当用户需要运行预设的天文教学课程时，使用此工具。

**参数：**
- `pipeline_file`: 流水线文件路径（YAML格式）
- `mode`: 执行模式（"auto"自动执行 或 "manual"手动确认）

**返回：**
执行结果、完成的步骤数、生成的文件列表

### 工具 2: create_pipeline - 创建自动化流水线

当用户要求创建新的教学课程时，使用此工具。

**参数：**
- `title`: 课程标题
- `topic`: 天文主题（星座/天体名称）
- `duration`: 课程时长（如 "15min"）
- `targets`: 目标天体列表
- `detail_level`: 详细程度（"basic"基础/"medium"中等/"deep"深度）

**返回：**
生成的流水线文件路径和预览

### 工具 3: point_and_speak - 实时指星+讲解

当用户需要即时讲解某个天体时，使用此工具。

**参数：**
- `target`: 天体名称（支持中文，自动查询坐标）
- `text`: 解说词
- `lecture_md`: 完整讲稿（可选）
- `ppt_slides`: PPT大纲（可选）

**返回：**
执行状态和生成的文件

### 工具 4: query_celestial - 查询天体信息

当用户询问某个天体的详细信息时，使用此工具。

**参数：**
- `target`: 天体名称（中文/拉丁/编号均可）

**返回：**
天体的详细信息（坐标、物理参数、观测建议等）

### 工具 5: generate_courseware - 生成课件

仅生成教学材料，不执行硬件控制。

**参数：**
- `topic`: 课程主题
- `lecture_md`: 讲稿内容
- `ppt_slides`: PPT大纲

**返回：**
生成的文件路径

## 词映射表使用

系统内置智能词映射，自动处理：

1. **同义词映射**
   - "参宿四" = "Betelgeuse" = "α Ori"
   - "仙女座大星云" = "M31" = "NGC 224"

2. **别名处理**
   - "北斗七星" → 自动识别为大熊座的一部分
   - "南斗六星" → 自动识别为人马座的一部分

3. **模糊匹配**
   - "猎户" → 自动匹配 "猎户座"
   - "M42" → 自动匹配 "猎户座大星云"

## 技术参数

### 串口配置
- 默认波特率: 115200
- 超时时间: 1秒
- 支持自动检测可用串口

### TTS 配置
- 默认引擎: Edge-TTS (在线)
- 备选引擎: pyttsx3 (离线)
- 默认语音: zh-CN-XiaoxiaoNeural

### 输出配置
- 讲稿目录: {baseDir}/output/
- 流水线目录: {baseDir}/pipelines/
- 支持时间戳文件命名

## 错误处理

1. **硬件未连接**: 自动降级到 Mock 模式（打印虚拟日志）
2. **天体未找到**: 返回相似天体建议
3. **TTS失败**: 降级到 pyttsx3 或打印文本
4. **流水线错误**: 跳过当前步骤，继续执行后续步骤

## 最佳实践

1. **创建流水线前**，先用 query_celestial 确认天体坐标
2. **长时间课程** 使用 mode: "manual" 以便控制节奏
3. **户外教学** 建议预先生成流水线并测试
4. **生成课件** 时使用 Markdown 格式，便于后续编辑

## 依赖项

- Python 3.8+
- pyserial (串口通信)
- edge-tts (在线TTS)
- pyttsx3 (离线TTS)
- python-pptx (PPT生成)
- PyYAML (流水线解析)
- skyfield/astroquery (天文计算)
