## 🚀 三步完成安装

### 第一步：复制文件（5分钟）

#### Windows用户：
```cmd
# 在C盘根目录创建项目文件夹
mkdir C:\astro-teach
cd C:\astro-teach

# 复制整个插件文件夹
# 将"extensions\astro-teach"文件夹复制到 C:\astro-teach\
```

#### Linux/Mac用户：
```bash
# 在home目录创建项目文件夹
mkdir ~/astro-teach
cd ~/astro-teach

# 复制整个插件文件夹
cp -r /path/to/extensions/astro-teach ./
```

**或者直接：**
- 将 `extensions/astro-teach` 整个文件夹
- 复制到你的项目根目录

### 第二步：运行配置脚本（1分钟）

#### Windows：
```cmd
cd C:\astro-teach\astro-teach
install.bat
```

#### Linux/Mac：
```bash
cd ~/astro-teach/astro-teach
chmod +x install.sh
./install.sh
```

**脚本会自动：**
- ✅ 检查系统环境

## ⚠️ 常见问题

### Q1: 插件未加载
**症状**: `openclaw plugins list` 看不到 astro-teach

**解决方案**:
```bash
# 检查配置
cat ~/.openclaw/openclaw.json | grep astro

# 如果没有，手动添加
openclaw config set plugins.entries.astro-teach.enabled true

# 重启
openclaw restart
```

### Q2: 找不到流水线
**症状**: `openclaw astro list` 显示"没有找到流水线文件"

**解决方案**:
```bash
# 检查目录
ls ~/.openclaw/workspace/pipelines/

# 如果为空，复制示例文件
cp astro-teach/examples/pipelines/* ~/.openclaw/workspace/pipelines/

# 或创建新目录
mkdir -p ~/.openclaw/workspace/pipelines
cp astro-teach/examples/pipelines/* ~/.openclaw/workspace/pipelines/
```

### Q3: 串口连接失败
**症状**: 串口设备无法连接

**解决方案**:
```bash
# 查看可用串口
openclaw astro serial

# Windows: 检查设备管理器中的COM口
# Linux: ls /dev/ttyUSB*
# Mac: ls /dev/cu.*

# 修改配置
openclaw config set plugins.entries.astro-teach.config.serialPort "你的串口"
```

### Q4: 配置验证失败
**症状**: `Config invalid` 错误

**解决方案**:
```bash
# 运行诊断
openclaw doctor --fix

# 或手动编辑
nano ~/.openclaw/openclaw.json
```
